Introduction

ADT

ADT is short for abstract data type. It is a mathematical model for data types. And its data objects and the specification of the operations on it are independent of the storage representation of data objects and the operations on the objects.


Array

Array usually be realized as an continuous memory areas. Intuitively, array is an ordered set composed of the index and value. For each defined index, there is a related value. In mathematics, it's called of correspondence or mapping.


Linked list

Linked list is a linear collection of data elements, in which elements can be placed anywhere in memory areas.


Tree

Tree is an important data structure. Intuitively, it means when organizing data, the data are related by the form of tree branches.


Graph

Graph is composed of two sets: a finite nonempty set composed by vertex and a finite set composed by edge. Sometimes we also can use G = (V, E) to represent a graph, the V(G) and E(G) respectively represent the graph G’s vertex set and edge set.


Sortings

In computer science, arranging in an ordered sequence is called "sorting".


Hash

Hashing is a technology, which can make query, insert and delete operations have a good expected performance. The hashing can be divided into static hashing and dynamic hashing.
